URL Shortener (Full Stack)

A URL Shortener Microservice API built using Node, Express, and MongoDB. URLs passed to the API are stored in the database with a corresponding Base62 encoding. The server then responds with the shortest URL available (the base URL + Base62 encoding). This URL can in turn be used to look up and redirect to the original URL.
